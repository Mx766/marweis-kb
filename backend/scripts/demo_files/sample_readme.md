## 2. 临床评价范围

本报告覆盖以下方面：

- **安全性能**：产品在预期用途下的安全性评估
- **有效性能**：产品是否达到预期的临床效果
- **风险收益分析**：权衡产品受益与潜在风险

## 3. 等同性论证要点

根据 NMPA《医疗器械临床评价技术指导原则》，等同性论证需对比：

1. 适用范围（适应证、适用人群、使用部位等）
2. 结构组成和设计
3. 性能指标
4. 生物学特性

> **注意**：以上内容仅为模板示例，实际报告需根据具体产品编写。
