# 器械注册部界面重设计 — 开发需求提示词

> **角色**: 项目经理 → 前端开发
> **目标**: 器械注册部全界面重设计，简约美观，交互性好
> **品牌色**: `#1e50ae`(主) / `#ffc001`(强调) / `#fb8c00`(重点)
> **设计原则**: 白色卡片 + 微妙阴影 + 品牌蓝点缀 + 流畅过渡动画

## 需求 A: 顶部导航栏重设计 (AppHeader)

**文件**: `frontend/src/components/layout/AppHeader.vue`

**现状**: 60px 高，纯白背景，搜索框是 el-input，导航用 el-menu，用户区基础下拉。

**目标效果**:
```
┌──────────────────────────────────────────────────────────────┐
│ [M] 迈瑞生知识库  │ 首页  管理后台  │  🔍搜索... ⌘K │ [张三▾]│
│     Marweis KB   │                 │               │        │
└──────────────────────────────────────────────────────────────┘
```

**具体要求**:
1. **高度**: 56px
2. **背景**: 毛玻璃效果 — `background: rgba(255,255,255,.92); backdrop-filter: blur(12px);`
3. **Logo**: 左边 "M" 方块(32x32, 品牌蓝背景, 白色字, 圆角7px) + 右侧两行文字("迈瑞生知识库" 14px/700 深色 + "Marweis KB" 10px 灰色)
4. **导航链接**: 不用 el-menu，直接用 `<router-link>` — 灰色字 `#5a6270`，hover 变品牌蓝 + 浅蓝背景，active 加粗 + 浅蓝背景，圆角 7px
5. **搜索框**: 不用 el-input 组件，用原生 `<input>` 包裹在 div 里 — 默认背景 `#f2f4f7` 宽 220px，focus 时背景变白、边框变品牌蓝、外发光 `0 0 0 3px rgba(30,80,174,.07)`、宽度扩展到 280px，右侧显示 "⌘K" 快捷键提示
6. **用户区**: 头像方块(30x30, 品牌蓝, 圆角7px) + 姓名 + 下拉箭头 — hover 背景 `#f2f4f7`
7. **用户下拉面板**: 顶部显示用户头像 + 姓名 + 部门/角色，下方"个人工作台"和"退出登录"选项，12px 圆角，加深阴影

## 需求 B: 工作流导航栏重设计 (WorkflowSidebar)

**文件**: `frontend/src/components/layout/WorkflowSidebar.vue`

**现状**: 已经是横向导航栏，但样式基础，用的是简单的 tab + 下拉。

**目标效果**:
```
┌──────────────────────────────────────────────────────────────┐
│ [M 器械注册部] │ 参考文件 9 │ 分类目录 3 │ 临床评价资料 ▾ │...│
└──────────────────────────────────────────────────────────────┘
```

**具体要求**:
1. **高度**: 46px，白色背景，底部 1px 边框
2. **部门标识**: 最左边品牌色圆角方块(6px圆角)，显示部门字母图标 + 部门名，白色字 12px/600
3. **分隔线**: 标识右边 1px 竖线 `#dedede`，高 22px
4. **模块标签**:
   - 文字色 `#5a6270`，13px
   - hover: 变品牌蓝 + 底部滑入蓝色指示条(2px高, `transform: scaleX` 动画)
   - active: 品牌蓝 + 加粗 + 指示条常驻
   - 右侧显示文档数量小标签(11px, 灰色背景圆角)
   - 有子分类的显示下拉箭头 ▼
5. **下拉面板**:
   - 白色背景，10px 圆角，`box-shadow: 0 12px 32px rgba(0,0,0,.1)`
   - 顶部小标题(11px 灰色大写) + 分隔线
   - 子分类项: hover 浅蓝背景 + 品牌蓝字，右侧显示文档数
   - 底部"查看全部 →"链接
   - **动画**: `<Transition name="dropdown">` — 入场 180ms ease-out，opacity 0→1 + translateY(-6px→0) + scale(.97→1)；出场 120ms ease-in 反向
6. **非器械注册部**: 无编号模块的部门显示所有非公共分类，而非空列表（已修）
7. **hover 延迟**: 鼠标离开 150ms 后才关闭下拉，防止误触

## 需求 C: 文档列表页重设计 (CategoryView)

**文件**: `frontend/src/views/CategoryView.vue`

**现状**: 简易行式列表，el-button 子分类切换，el-input 搜索框。

**目标效果**:
```
┌──────────────────────────────────────────────────────────────┐
│ 首页 › 参考文件                                               │
│ 参考文件                                                     │
│ 通用法规及指导原则参考             9 篇文档                    │
│                                                              │
│ [全部] [通用法规(国内)] [通用法规(国外)] [指导原则] [审评论坛]  │
│                                                              │
│ 🔍 在当前分类下搜索文档...              [全部格式 ▾]          │
│                                                              │
│ ┌──────────────────────────────────────────────────────────┐ │
│ │ [PDF] 医疗器械监督管理条例（2025修订）          法规 注册  │ │
│ │       张三 · 2026-07-14 · v2025          👁 127          │ │
│ │       明确了医疗器械全生命周期监管要求...                  │ │
│ ├──────────────────────────────────────────────────────────┤ │
│ │ [DOC] 医疗器械注册管理办法                  注册 管理     │ │
│ │       ...                                                │ │
│ └──────────────────────────────────────────────────────────┘ │
│                      < 1  2  3 ... 10 >                      │
└──────────────────────────────────────────────────────────────┘
```

**具体要求**:
1. **分类头部**: 
   - 面包屑用 `›` 分隔符
   - 标题 22px/700 深色
   - 描述文字 13px 灰色 + 文档数量(数字 16px/700 品牌蓝)
2. **子分类** 用圆角 `pill` 按钮(20px 圆角):
   - 默认: 白底灰边框灰字
   - hover: 品牌蓝边框 + 品牌蓝字
   - active: 品牌蓝填充 + 白字加粗
3. **搜索框**: 不用 el-input，用原生 `<input>` 包裹在白色 div 里:
   - 左侧搜索图标 + 右侧清除按钮
   - 10px 圆角
   - focus 时边框变品牌蓝 + 外发光
   - **输入 300ms 防抖自动搜索**（不用手动回车）
4. **文件格式筛选**: el-select，简化选项为 "全部格式 / PDF / Word / Excel / 外链"
5. **文档卡片**:
   - 白色背景，hover 变 `#fafcff`，active 变 `#f0f4fa`
   - 左侧文件类型色块(48x48, 10px圆角):
     - PDF: `#fef0f0` / 字色 `#e74c3c`
     - Word: `#eef4fd` / 字色 `#1e50ae`
     - Excel: `#edf7ee` / 字色 `#27ae60`
     - 外链: `#e6f7f9` / 字色 `#00bcd4`
   - 中间: 标题(14px/600) + 元信息(12px灰色) + 摘要(12.5px灰色, 最多2行)
   - 右侧: 标签(11px灰色背景) + 浏览量
6. **空状态**: 大图标 + "暂无文档" + 提示文字
7. **响应式**: 768px 以下隐藏右侧标签列，搜索框全宽
8. **页面最大宽度**: 960px 居中

## 通用交互规范

| 交互 | 规范 |
|------|------|
| **hover 过渡** | 150ms ease，背景/边框/颜色变化 |
| **下拉面板** | 180ms ease-out 入场，120ms ease-in 出场 |
| **搜索防抖** | 300ms |
| **卡片点击** | router.push 导航 |
| **空状态** | 大图标 + 标题 + 描述文字 |
| **加载态** | el-loading 或骨架屏 |
| **响应式断点** | 768px |
